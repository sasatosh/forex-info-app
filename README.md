# 為替レート情報アプリ

リアルタイムで為替レート（TTM・TTB・TTS）を取得・表示するWebアプリケーションです。

## 機能

- **TTM/TTB/TTS表示**: 仲値・買相場・売相場を分けて表示
- **日付指定**: 過去の日付のレートも取得可能
- **主要10通貨対応**: USD, JPY, EUR, GBP, AUD, CAD, CHF, CNY, KRW, INR
- **基準通貨の切り替え**: 任意の通貨を基準に設定可能
- **履歴データ対応**: Frankfurter APIを使用して過去のレートを取得
- **レスポンシブデザイン**: モバイル・タブレット・デスクトップ対応

## 対応通貨

- USD（米ドル）
- JPY（日本円）
- EUR（ユーロ）
- GBP（英ポンド）
- AUD（豪ドル）
- CAD（加ドル）
- CHF（スイスフラン）
- CNY（中国元）
- KRW（韓国ウォン）
- INR（インドルピー）

## レート種類

### TTM（Telegraphic Transfer Middle Rate / 仲値）
銀行が公示する基準レート。実際の取引では使用されず、TTBとTTSの中間値です。

### TTB（Telegraphic Transfer Buying Rate / 買相場）
銀行が外貨を買い取る（顧客が外貨を売る）際のレート。TTMよりも低い値となります。

### TTS（Telegraphic Transfer Selling Rate / 売相場）
銀行が外貨を販売する（顧客が外貨を買う）際のレート。TTMよりも高い値となります。

**例）** USD/JPY = 150.00円（TTM）の場合
- TTB: 149.00円（顧客がドルを売る時）
- TTS: 151.00円（顧客がドルを買う時）

※ スプレッド（手数料）は通貨ごとに異なります

## 技術スタック

- **フロントエンド**: React 19
- **ビルドツール**: Vite 7
- **API**:
  - ExchangeRate-API（最新レート取得）
  - Frankfurter API（履歴データ取得）
- **スタイリング**: CSS3

## セットアップ

### 必要な環境

- Node.js 18以上
- npm または yarn

### インストール

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ブラウザで http://localhost:5173/ にアクセス
```

### ビルド

```bash
# 本番用ビルド
npm run build

# ビルド結果のプレビュー
npm run preview
```

## 使い方

1. **基準通貨を選択**: ドロップダウンから基準通貨を選択（デフォルト: JPY）
2. **日付を選択**: カレンダーから日付を選択（過去のレートも取得可能）
3. **レートの確認**: 各通貨のTTM・TTB・TTSが自動的に表示されます
4. **手動更新**: 「🔄 更新」ボタンでレートを再取得できます

### 表示例

**JPYを基準通貨として選択した場合:**
- USD: TTM 150.00円、TTB 149.00円、TTS 151.00円
- EUR: TTM 165.00円、TTB 163.50円、TTS 166.50円

**USDを基準通貨として選択した場合:**
- JPY: TTM 0.0067、TTB 0.0066、TTS 0.0068
- EUR: TTM 1.10、TTB 1.09、TTS 1.11
